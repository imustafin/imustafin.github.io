Dictionary

This is the main page of my very subjective English-Russian-Tatar dictionary of spoken language.

This main objective of this dictionary is to collect both real academic translations of words as well as jargons. The words collected here are mainly in the field of Software Engineering.

Words

  • Asset Pipeline (Russian: Файлопровод, Tatar: Файлүткәргеч) — Framework to concatenate and compress JS, CSS and other assets in Ruby on Rails
  • Cache (Russian: Кеш, Tatar: Кәш) — An auxillary memory which from which data can be read faster than from the main data source
  • Hook (Russian: Хук, Tatar: Һук) — Code which alters or augments the behaviour of a program by intercepting messages or events
  • To cache (Russian: Кешировать, Tatar: Кәшләргә) — To save data to cache
